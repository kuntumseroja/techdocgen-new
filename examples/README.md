# Examples

This folder contains small, self-contained projects that exercise the new
capabilities:

- `angular-app/`: TypeScript + HTML (Angular-style structure)
- `node-service/`: Node.js + TypeScript + RabbitMQ client usage
- `csharp-masstransit/`: C# MassTransit/RabbitMQ flows
- `infra-config/`: JSON/YAML infrastructure configuration
